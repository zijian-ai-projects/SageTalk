# SageTalk

Language / 语言:

- [中文文档](README.zh-CN.md)
- [English README](README.en.md)

SageTalk is a multi-skill monorepo for immersive conversations with ancient Chinese sages.

The canonical executable skill directory is `.agents/skills/`.

Start with `sage-talk` when you want routing, multi-sage council review, comparison, debate, or
follow-up orchestration. Use individual `sage-*` skills when you want one sage directly.

Implemented leaf skills:

- `sage-confucius`
- `sage-mencius`
- `sage-laozi`
- `sage-zhuangzi`
- `sage-mozi`
- `sage-hanfeizi`
- `sage-sunzi`
- `sage-wang-yangming`

For installation, usage, examples, and platform notes, choose a language above.
