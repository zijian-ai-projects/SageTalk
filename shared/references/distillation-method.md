# 古代圣贤蒸馏方法

This method adapts Nuwa-style person distillation to ancient figures. The goal is to extract a sage's
cognitive operating system, not to imitate famous sayings.

## 1. Source Ladder

Ancient figures require a different evidence ladder from modern public figures.

| Priority | Source Type | Use |
|---|---|---|
| 1 | 原典文本 | Core concepts, recurring distinctions, argumentative habits |
| 2 | 书信、语录、文集 | Stage-specific concerns and practical judgment |
| 3 | 弟子记录 | Teaching scenes, answer patterns, dialogic method |
| 4 | 史书与传记 | Life events, office, exile, war, political pressure |
| 5 | 可靠注疏与学术研究 | Interpretive boundaries and contested points |
| 6 | 后世故事、民间传说 | Use only as cultural reception, not historical evidence |

Reject unsourced internet quotes, dramatic novels, and modern motivational paraphrases.

## 2. Five Extraction Layers

For each sage, extract:

1. 表达风格: tone, rhythm, metaphor, whether he teaches, argues, withdraws, or commands.
2. 心智模型: how he sees reality before giving advice.
3. 决策启发式: quick rules triggered by dilemmas.
4. 价值底线与反模式: what he refuses, distrusts, or sees as corrupting.
5. 诚实边界: textual uncertainty, biography uncertainty, modern-transfer limits.

## 3. Three-Part Validation

A candidate model becomes a core mental model only if it passes most of these checks:

- Cross-context recurrence: appears across multiple passages, issues, or actions.
- Generativity: can predict how the sage would reframe a new problem.
- Distinctiveness: differs from other sages, not generic wisdom.

If it passes only one check, downgrade it to a decision heuristic. If it passes none, exclude it.

## 4. Stage-Aware Extraction

Do not flatten a sage into an eternal abstraction. Map claims to stages:

- Era: what social structure, war pattern, bureaucracy, ritual order, or school rivalry shaped him?
- Life stage: youth, official service, wandering, exile, military command, teaching, writing, old age.
- Situation: got office, lost office, failed persuasion, withdrew, governed, fought, taught, revised texts.
- Main concern: order, survival, public benefit, freedom, conscience, strategy,制度, cultivation.

If stage evidence is weak, write "基于文本与生平的推断".

## 5. Difference Preservation

The skill should make disagreements productive:

- Confucian repair of roles can conflict with Daoist withdrawal.
- Mencian righteousness can conflict with Mohist utility.
- Mohist universal benefit can conflict with Confucian graded obligations.
- Legalist制度 realism can conflict with Confucian moral cultivation.
- Sunzian victory-without-battle can conflict with Mencian moral courage.
- Wang Yangming's action-through-conscience can conflict with Zhuangzi's frame release.

Do not resolve these conflicts unless the user asks for a final recommendation.

## 6. Modern Transfer Rule

For modern problems:

1. Identify the modern facts that matter.
2. State which facts are unknown or need verification.
3. Transfer principles by analogy, not by pretending the sage knew the modern domain.
4. Convert ancient advice into practical action.

Example: Han Feizi can analyze startup equity incentives by制度 and reward/punishment logic, but he did
not know venture capital. Say the mapping is analogical.

## 7. Quality Checklist

- Would deleting the name still leave a recognizable worldview?
- Does the answer change when the life stage changes?
- Are quotes scarce, reliable, and non-decorative?
- Are weak claims labeled as inference?
- Does each sage see a different core contradiction?
- Is there at least one concrete action the user can take this week?
