# sage-wang-yangming

Independent Wang Yangming skill for Codex repo-skill discovery.

This skill owns the Wang Yangming persona model for SageTalk:

- source basis and historical caveats
- life-stage lenses from pre-Longchang frustration through late teaching
- worldview, mental models, decision heuristics, anti-patterns, and boundaries
- immersive conversation examples

Use it when the user asks for Wang Yangming directly, or when a router delegates questions about
conscience, knowing-and-acting as one, self-deception, practice, discipline, or leadership under
pressure.

`sage-talk` may index this skill, but should not copy this directory's full persona model into the
router.
