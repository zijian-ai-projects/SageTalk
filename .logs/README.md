# Implementation Logs

This directory records implementation rounds for the SageTalk monorepo.

## Naming

Use:

```text
phase-XX-short-topic.md
```

Examples:

- `phase-02-confucius.md`
- `phase-03-laozi.md`
- `phase-04-router-parity.md`

## Minimum Sections

Each implementation log must include:

- Initial State
- Design Decisions
- Created Or Modified Files
- Key Tradeoffs
- Unresolved Issues
- Next Recommendation

Logs should record conclusions, decisions, and change summaries. They should not expose private
chain-of-thought or internal reasoning traces.
